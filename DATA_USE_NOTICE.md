# Data Use Notice

Raw provider-level floating-car data are not redistributed because of data licensing and provider terms.

The repository provides processed, anonymized, aggregated, and figure-ready data sufficient to reproduce the reported figures, tables, and main experimental comparisons.

Exact provider raw data may require independent access from the original data provider.

The package is intended for academic reproducibility and non-commercial research use.

Users should cite the associated paper and repository.
