# Pan Borneo Sparse FCD Forecasting - Paper Data Package

This package contains cleaned/derived/aggregated reproducibility-supporting data, figure/table candidate source data, metrics, statistics, validation reports, logs, and configs for GPT review. It excludes restricted third-party raw provider data. Figures and final manuscript tables are not rendered here.

Reproduce checks with the validation/audit scripts in the source project. Citation and contact: placeholders.
