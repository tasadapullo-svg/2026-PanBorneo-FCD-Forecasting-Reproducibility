# Public upload notice

This folder is a conference-paper support package generated from the verified Pan Borneo sparse FCD forecasting outputs.

Included:
- validated metrics, prediction parquet files, statistical tests, logs, configs, reports, and GPT-review candidate data;
- Phase 13, Phase 15, Phase 16, and Phase 20 support files;
- GitHub/Zenodo preparation metadata and checksums.

Excluded:
- restricted third-party raw provider data;
- local conda environments, caches, credentials, API keys, and machine-specific temporary files.

Important warnings:
- METR-LA-mini is supplementary only and must not be mixed into Pan Borneo main figures/tables.
- In Phase 20, Proposed is a TCN alias for external sanity checking; do not claim Proposed outperforms TCN on METR-LA-mini.
- This package does not render final manuscript figures or final manuscript tables.
